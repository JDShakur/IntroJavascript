Laços de Repetição

Quando necessitamos executar uma tarefa inúmeras vezes utilizamos os sempre úteis laços de repetição, no JavaScript há tem formas de fazer isso acontecer, sendo elas: "For","While" e "Do while", essas são as principais ferramentas de para repetição, não utilizado somente de js mas também de diversas linguagens.

Loop 'For'

Aqui teremos um exemplo, começando pela sintaxe do Loop For 

                    
                        for (inicialização; condição; expressão de iteração) {
                            // Código a ser repetido
                          }
                          //de outra maneira veremos:
                        for(let i = 0;i <= 10;i++){
                            código
                        }

Nisso vemos que na estrutura do For as números de iterações devem ser impostas antecipadamente, isso dentro da colchetes da função. Ela também pode ser dividida em três partes, sendo elas: A Inicialização (representado como o let i = 0), a vondição (nesse caso i <= 10) e a expressão da interação(já que é uma variável crescente i++);

Loop 'while'

Outra loop de repetição (e meu favorito) é o loop while. Extremamente fácil de se usar, e preferivel em muitoscasos no qual você tem a variável em uma condição e repetição já estabelecida, fazendo que a condiçãode verificação seja feita antes de cada interação, caso não atenda mais oque o loop exige o loop é encerrado retornando algo.

Aqui vaia sintaxe do while:

                            
                                while(condição){
                                    código a ser repetido
                                }
                                // exemplo detalhado
                                let i = 0;
                                while(i <=10){
                                    console.log(i);// Imprime os números de 0 a 10.
                                    i++
                                }

Loop 'Do while'

Apesar do nome ser parecido, o loop Do While funciona de uma forma diferente do While, Onde temos a condição sendo verificada após o código, garantindo que o código 'rode' pelo menos 1 vez antes de atender as condições.

Vamos a sintaxe e para um exemplo de código: 

                            
                                //sintaxe
                                do {
                                    // Código a ser repetido
                                  } while (condição);
                                  //exemplo
                                  let i = 0;
                                    do {
                                    console.log(i); // Imprime os números de 0 a 4
                                    i++;
                                    } while (i < 5);

Para facilitar a visão de como funciona este loop, podemos imaginar um caixa eletrônico, que executa o layout de funções('while' ou seja enquanto ele não decide), antes da decisão do usuário de sacar, depositar, tirar extrato etc('do' após ser selecionado).
